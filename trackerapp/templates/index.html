What's SI Issue Tracker?

SI Issue Tracker is a simple but powerful issue tracker app created as part of a Python/Django course project. Users of the app will be able to, amongst other things, create an account and manage that account, and also submit issues, track issues, scale issues, submit features, and track those features, etc. Users Can:

  • Get Set Up Within Minutes
  • View Statistics
  • Upscale or Downscale Issues
  • Upscale or Downscale Features
